# Stock Analysis MCP Server

Connects Claude to live stock data via Yahoo Finance.
Provides 5 tools: price history, technical indicators,
fundamentals, investment scenarios, and peer comparison.

## Setup
1. pip install -r requirements.txt
2. Register with Claude Code (see guide)
3. Ask Claude: "Run a full analysis on NVDA with $400 invested for 12 weeks"
